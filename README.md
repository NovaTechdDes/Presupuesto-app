# Presupuesto App

## Instalacion de dependencias

1. Ejectutar `pnpm install`
2. Ejectutar `pnpm  expo start`

## Construccion del build en expo

1. Ejecutar `npx expo-doctor`
2. Ejectura `npx expo install --check`
3. Ejecutar `eas build -p android --profile preview`

## Actualizacion

1. Ejectutar `eas update --branch preview --message "Primer update OTA"`
